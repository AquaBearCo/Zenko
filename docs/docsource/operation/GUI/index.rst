.. _`XDM UI`:

Using the XDM UI
================

The XDM UI is an application-based graphical user interface for the XDM
platform. While XDM offers its full functionality through a rich API, many
of its features are also available through this graphical user interface,
providing ease of use and excellent opportunities for visualizing managed data.
